O Mago que Não Sabia Fazer Magia

No Reino dos Encantados, existia um mago chamado Rufus, que não tinha o menor talento para magia. Mas ele adorava se exibir: "Eu sou o mago mais poderoso de todos os tempos!", ele dizia, tentando fazer uma poção que, em vez de curar, transformava a água em suco de laranja... bem ácido. Mas ninguém ousava questioná-lo. Certa vez, ele tentou transformar uma pedra em ouro. Mas, claro, ele acabou criando uma pedra um pouco mais brilhante do que o normal. Orgulhoso, ele anunciou: — Olhem! Eu criei o diamante mais belo de todos os tempos! — Uau, mestre Rufus, qual é o truque? — Nenhum truque. Só não sei mais o que fazer com isso. Tá bom para uma decoração, né?
